### IGDB API setup:

In order to get the API Client ID and Secret, follow the steps outlined within the [documentation](https://api-docs.igdb.com/#account-creation). After registring the application, you can find the Client ID and Secret under the application details.

Finally, follow the [instructions](IaC\README.md) on how to create the Secrets within the Azure Key-Vault to securely store the API-credentials.
